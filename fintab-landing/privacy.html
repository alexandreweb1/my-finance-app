Política de Privacidade

Fintab — Controle Financeiro Pessoal

Última atualização: 30 de março de 2026

Esta Política de Privacidade descreve como o aplicativo Fintab, desenvolvido por Alexandre Figueiredo Dev (disponível na Google Play Store sob o ID com.alexdev.myfinanceapp), coleta, usa e protege as informações dos seus usuários.

1. Informações que coletamos

O Fintab coleta apenas as informações necessárias para o funcionamento do app:

  • Dados de autenticação: nome, endereço de e-mail e foto de perfil fornecidos via login com Google ou cadastro por e-mail/senha.
  • Dados financeiros: transações, categorias, carteiras e orçamentos que o próprio usuário registra voluntariamente no app.
  • Dados de assinatura: informações sobre o plano Pro (tipo, status e data de validade), processadas pela Google Play Store.
  • Notificações de outros apps (opcional): quando o usuário ativa a funcionalidade "Detectar transações", o app lê notificações de apps bancários instalados no dispositivo para sugerir lançamentos automaticamente. Nenhum conteúdo de notificação é transmitido a servidores externos.

2. Como usamos as informações

  • Autenticar e identificar o usuário na conta.
  • Sincronizar dados financeiros entre dispositivos via Firebase (Google Cloud).
  • Verificar e gerenciar assinaturas Pro.
  • Sugerir lançamentos automáticos a partir de notificações lidas localmente no dispositivo.

3. Armazenamento e segurança

Os dados são armazenados no Firebase Firestore (Google Cloud), com acesso restrito à conta do próprio usuário por meio de regras de segurança. Utilizamos criptografia em trânsito (HTTPS/TLS) em todas as comunicações.

Dados de notificações lidos para a funcionalidade de detecção automática são processados exclusivamente no dispositivo e nunca enviados a servidores externos.

4. Compartilhamento de dados

Não vendemos, alugamos nem compartilhamos dados pessoais com terceiros para fins comerciais. Os dados podem ser acessados pelos seguintes provedores de infraestrutura, exclusivamente para fins operacionais:

  • Google Firebase — autenticação e banco de dados.
  • Google Play Store — processamento de pagamentos e assinaturas.

5. Dados de menores

O Fintab não é direcionado a crianças menores de 13 anos e não coleta intencionalmente dados de menores. Caso identifique que dados de uma criança foram coletados, entre em contato para que possamos removê-los imediatamente.

6. Seus direitos (LGPD / GDPR)

Você tem direito a:

  • Acessar os dados pessoais que temos sobre você.
  • Corrigir dados incorretos ou desatualizados.
  • Solicitar a exclusão de todos os seus dados.
  • Revogar o consentimento a qualquer momento.

Para exercer qualquer desses direitos, entre em contato pelo e-mail: alexandre.filho.dev@gmail.com

7. Exclusão de dados

Para solicitar a exclusão completa da sua conta e todos os dados associados, envie um e-mail para alexandre.filho.dev@gmail.com com o assunto "Exclusão de dados — Fintab". O prazo para conclusão é de até 30 dias.

8. Alterações nesta política

Reservamo-nos o direito de atualizar esta política periodicamente. Mudanças significativas serão comunicadas dentro do próprio app. O uso continuado do Fintab após as alterações implica na aceitação da política revisada.
